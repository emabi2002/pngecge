PNGEC-BRS Database Setup

Follow these steps to set up your Supabase database

1

Copy the Database Schema

Click the button below to copy the complete SQL schema to your clipboard.

Copied!
Loading schema...
2

Open Supabase SQL Editor

Click the button below to open the Supabase SQL Editor in a new tab.

Open SQL Editor
3

Paste and Run

In the SQL Editor:

1. Clear any existing content in the editor

2. Press Ctrl+V (or Cmd+V on Mac) to paste the schema

3. Click the green "Run" button

4. Wait for the success message (usually 10-20 seconds)

4

Seed Initial Data

After the schema is applied, click the button below to seed the database with initial data.

cd pngec-brs && bun run db:seed

5

Verify Setup

After seeding, refresh your PNGEC-BRS application to see live data.
